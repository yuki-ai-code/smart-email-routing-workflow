# Gmail AI Automation

## Overview

This project is an AI automation workflow built using n8n and Gmail API.

The workflow retrieves Gmail messages and processes them for AI-based automation.

---

## Technologies Used

- n8n
- Gmail API
- OAuth2
- OpenAI API
- JSON

---

## Workflow Structure

Gmail Trigger → AI Processing → Notification / Automation

---

## Features

- Gmail message retrieval
- OAuth2 authentication
- AI automation workflow
- Gmail API integration
- No-code automation using n8n
